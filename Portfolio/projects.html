This project was for my school project, the idea behind the project was to create a trivia game. The project was done on replit https://replit.com/@BenjaminRampers/PythonProjectTriviaGame
I used the Pil library to make a png, which I have displayed as well as using the console for the bulk of the game.
The display png aptly named display.png, is changed based on what you're doing in the game and how well you're doing.
The main game code(the code that gives you the questions and you interact with) is put inside a while loop.
Inside the loop, I used multiple functions which made it easier for organizing and to have less repeats of the code.
The questions were made using objects, and put inside an array, Some of the categories have no questions.
I also wanted to let the player make custom questions, it was done by having a loop for the amount of questions to be input and putting the object immediately in the array.
I was going to add files that would be saved but in the interest of simplicity they have none
